# HTML-05 : Modal Popup using CSS (:target)

## Objective
Build a modal (popup window) that opens and closes without using JavaScript, using the CSS `:target` pseudo-class.

---

## What I Implemented

- Created a modal popup triggered using an anchor link (`href="#modal"`)
- Used the `:target` pseudo-class to control modal visibility
- Designed a full-screen overlay using `position: fixed`
- Centered the modal using Flexbox
- Added smooth transitions using:
  - `opacity`
  - `visibility`
  - `transform`
- Implemented a close button that removes the target (`href="#"`)
- Ensured clean UI with a backdrop and popup animation
