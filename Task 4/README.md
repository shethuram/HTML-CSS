# HTML-04 : Responsive Navbar with Dropdown

## Objective

Build a responsive navigation bar with a dropdown menu and integrate it with a grid-based layout.

---

## What I Implemented

* Created a navigation bar using `ul` and `li`
* Added a dropdown menu under "Events"
* Implemented dropdown visibility using CSS `:hover`
* Positioned dropdown using `position: absolute` inside a relative parent
* Styled dropdown items with spacing and background color
* Dropdown menu adjusted using `position: static`
